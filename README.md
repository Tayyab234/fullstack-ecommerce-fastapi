# FullStack E-Commerce Platform

A modern full-stack eCommerce web application built using React, TypeScript, FastAPI, and MongoDB.

This project includes dynamic product loading, cart management, order handling, tracking functionality, and backend-driven architecture.

# Backend Setup

Navigate to backend directory:

```bash
cd backend
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run backend server:

```bash
uvicorn backend:app
```

Backend will run on:

```bash
http://127.0.0.1:8000
```

---

# Database Initialization

The backend includes:

- `products.json`
- `init.py`

These files are used to initialize and store products inside MongoDB.

Run initialization script if needed:

```bash
python init.py
```

---

# Frontend Setup

Navigate to frontend directory:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Run frontend:

```bash
npm run dev
```
